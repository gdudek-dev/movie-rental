## General info
Project in which I learn Spring Boot.  
* Spring MVC is used as design pattern.  
* Models are based on "sakila" database.  
* Views are created with HTML5, CSS and Thymeleaf.


## Technologies
Project is created with:
* Java: 11
* Maven
* Hibernate
* Spring Boot
* Spring Security
* Spring MVC
* Thymeleaf
* HTML5
* CSS
